# 📊 BÁO CÁO KẾT QUẢ THỰC HIỆN (DAILY & WEEKLY REPORT)

Thư mục này chứa **kết quả thực tế** của từng ngày và từng tuần — điền sau khi kết thúc ngày.

---

## Tại sao cần tách Report riêng?

- **Plan** = bản đồ trước khi lên đường
- **Report** = nhật ký ghi chép những gì thực sự đã xảy ra
- Tách riêng giúp nhìn rõ **khoảng cách giữa kế hoạch và thực tế** — đây là dữ liệu quý nhất để cải thiện.

---

## Nội dung file Report ngày

Mỗi file Report bao gồm:
- ✅ **Checklist hoàn thành** — từng task theo ABCDE
- ⏱️ **Thực tế so với kế hoạch** — làm đúng giờ không?
- 🏆 **3 điều tốt hôm nay** (Thói quen #1)
- 📈 **Điểm tự đánh giá (1-10)**
- 💡 **Bài học / điều cần cải thiện**
- 🔥 **Streak thói quen**

---

## Quy tắc đặt tên

```
YYYY-MM-DD-Report-[Ten-Ngay]-Tuan-[So].md      ← Report ngày
YYYY-W[So-Tuan]-Weekly-Report.md               ← Report tuần
```

Ví dụ:
- `2026-07-19-Report-Thu-Bay-Tuan-3.md`
- `2026-W30-Weekly-Report.md`

---

## Cấu trúc thư mục

```
Report/
└── 07-2026/
    ├── 2026-07-19-Report-Thu-Bay-Tuan-3.md
    ├── 2026-07-20-Report-Chu-Nhat-Tuan-3.md
    └── Weekly/
        ├── 2026-W29-Weekly-Report.md
        └── 2026-W30-Weekly-Report.md
```

---

## Liên kết với loại file khác

| Loại | Thư mục | Nội dung |
|------|---------|---------|
| **Plan** | `Plan/Daily/` | Kế hoạch trước khi làm |
| **Journal** | `Journal/` | Nhật ký cảm xúc, phản tư |
| **Report** (file này) | `Report/` | Kết quả thực tế đã làm |

---

*Không đo lường = không cải thiện. Report là gương phản chiếu sự thật.*
