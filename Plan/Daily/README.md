# 📅 KẾ HOẠCH NGÀY CHI TIẾT (DAILY PLAN)

Thư mục này chứa **bản kế hoạch chi tiết từng ngày** — viết trước khi bắt đầu ngày mới.

---

## Nội dung file Plan

Mỗi file Plan bao gồm:
- 🐸 **Con ếch ngày hôm nay** — task quan trọng nhất (A1)
- 📋 **Bảng ABCDE** — phân loại toàn bộ công việc
- 🕐 **Lịch trình chi tiết** — từng khung giờ
- ⚔️ **Thách thức dự kiến** — và cách vượt qua
- ✅ **Định nghĩa "Ngày Thành Công"** — mức tối thiểu / tốt / xuất sắc
- 📊 **Checklist kết quả cuối ngày** — điền sau khi kết thúc ngày

---

## Quy tắc đặt tên

```
YYYY-MM-DD-Plan-[Ten-Ngay]-Tuan-[So].md
```

Ví dụ:
- `2026-07-19-Plan-Thu-Bay-Tuan-3.md`
- `2026-07-20-Plan-Chu-Nhat-Tuan-3.md`

---

## Cấu trúc thư mục

```
Plan/Daily/
└── 07-2026/          ← nhóm theo tháng
    ├── 2026-07-11-Plan-Thu-Bay-Tuan-1.md
    ├── 2026-07-12-Plan-Chu-Nhat-Tuan-1.md
    └── ...
```

---

## Liên kết với loại file khác

| Loại | Thư mục | Nội dung |
|------|---------|---------|
| **Plan** (file này) | `Plan/Daily/` | Kế hoạch trước khi làm |
| **Journal** | `Journal/` | Nhật ký cảm xúc, phản tư |
| **Report** | `Report/` | Kết quả thực tế đã làm |

---

*Kế hoạch là bản đồ. Hành động mới là cuộc hành trình thật sự.*
